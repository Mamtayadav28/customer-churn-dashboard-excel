# 📊 Customer Churn Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes customer churn behavior using an interactive Excel dashboard.

---

## 📊 Dashboard Preview
![Dashboard](customer_churn_dashboard.xlsx)

---

## 🎯 Objectives
- Identify high-risk customers
- Understand churn behavior
- Provide business insights

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Slicers

---

## 📈 Key Insights
- Customers inactive for >60 days are more likely to churn
- Complaints significantly increase churn risk
- Discounts have limited impact on retention

---

## 🚀 How to Use
1. Download Excel file
2. Open in Excel
3. Use filters to explore dashboard
